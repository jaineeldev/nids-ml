# data/

This folder contains all data files used in the project.

## NSL-KDD Dataset

**File:** `KDDTrain+.txt`  
**Source:** https://github.com/defcom17/NSL_KDD  
**Purpose:** Labelled network traffic dataset for ML model training

### Columns
The dataset has 42 columns. The second-to-last column is the **label** (e.g. `normal`, `neptune`, `smurf`) and the last column is the **difficulty score**.

### Attack categories
| Label | Type |
|---|---|
| normal | Normal traffic |
| neptune, smurf, pod | DoS attacks |
| ipsweep, portsweep, nmap | Probe attacks |
| ftp_write, guess_passwd | R2L attacks |
| buffer_overflow, rootkit | U2R attacks |

## Captured Data

Files named `capture_YYYY-MM-DD_HH-MM-SS.csv` are live captures generated by `main.py` — these are gitignored and not pushed to GitHub.

`processed.csv` is the cleaned and encoded version of the latest capture — also gitignored.
