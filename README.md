# nids-ml

> 🚧 **Project in active development** — check back for updates.

ML-powered network intrusion detection system built in Python. Uses machine learning to classify network traffic as normal or malicious in real time.

---

## Overview

This project combines cybersecurity and machine learning to detect network intrusions and threats. Built as a collaboration between a Cybersecurity student and a Machine Learning engineer.

## Planned Features

- [ ] Live packet capture and analysis
- [ ] ML model to classify network traffic
- [ ] Real-time alert dashboard
- [ ] Support for the NSL-KDD dataset
- [ ] REST API for model inference

## Tech Stack

| Area | Tools |
|---|---|
| Language | Python 3.11+ |
| Packet Capture | Scapy |
| ML / Data | Scikit-learn, Pandas, NumPy |
| API | Flask |
| Dataset | NSL-KDD |
